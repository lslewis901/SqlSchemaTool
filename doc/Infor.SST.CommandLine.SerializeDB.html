SQLServer
The string for the SQL server name to connect to.
DBName
The string for the SQL server database name to connect to.
UID
The string for the SQL server user login to use for the SqlConnections object.
PWD
The string for the SQL server user password to use for the SqlConnections object.
SQLfile
The SQL script file that will be translated from the create XML schema file.
Translate
A flag to cause the translation method to be called, thus creating the SQL script file specified with the SQLfile param.
Primary
A flag to force the FileGroups to only have Primary, no matter what the actual one is in the database.

Return Value

Returns the Serialized XML output filename
