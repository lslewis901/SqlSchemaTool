Infor.SST.SQLObjects Namespace

The Infor.SST.SQLObjects namespace classes are used to provide a objects to serialize SQL server database objects.

These objects consist of defaults, UDDTs, rules, tables, table columns, functions, stored procedures, triggers, and eventually jobs.
