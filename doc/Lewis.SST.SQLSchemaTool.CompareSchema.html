The destination/target XML file name to compare with the source XML file.
diffgramFile
The XML Diffgram file name to create.
SQLfile
The SQL script file that will be translated from the diffgram XML schema file.
CompareSprocText
A flag to cause stored procedures to be compared character by character, excluding whitespace, tabs, linefeeds and carriage returns.
CompareViewText
A flag to cause views to be compared character by character, excluding whitespace, tabs, linefeeds and carriage returns.
Translate
A flag to cause the translation method to be called, thus creating the SQL script file specified with the SQLfile param.
objectsToSerialize
bit mask of SQL objects to serialize - use the _nodetype enum
CustomXSLT
the full path and name of a custom xslt to be applied to the final xml output
