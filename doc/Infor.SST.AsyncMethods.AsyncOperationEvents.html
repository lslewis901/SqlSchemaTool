AsyncOperation Events

The events of the AsyncOperation class are listed below. For a complete list of AsyncOperation class members, see the AsyncOperation Members topic.

Public Instance Events

Cancelled This event will be fired when the operation is successfully stoped through cancellation. This event will be raised through the ISynchronizeTarget supplied at construction time.
Completed This event will be fired if the operation runs to completion without being cancelled. This event will be raised through the ISynchronizeTarget supplied at construction time. Note that this event may still be received after a cancellation request has been issued. (This would happen if the operation completed at about the same time that cancellation was requested.) But the event is not raised if the operation is cancelled successfully.
Failed This event will be fired if the operation throws an exception. This event will be raised through the ISynchronizeTarget supplied at construction time.
