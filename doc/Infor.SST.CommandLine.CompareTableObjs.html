CommandLine.CompareTableObjs Method 

Compares table indexes, table constraints, and table FK/PK references from two distinct XML schema files. These files are generated by the SerializeDB method. This method is called by the CompareTables method.

   XmlNode DiffTable,
   XmlDocument xmlDiffDoc,
   XmlDiff xmlDiff
);

Parameters

objType
The type of database table related object collection to compare, i.e. indexes, constraints (check and default), and references.
SourceTable
The SourceTable XML Node containing all table object related child nodes to compare with the destination child nodes.
DestTable
The DestTable XML Node containing all table object related child nodes to compare with the source child nodes.
DiffTable
The XML Diffgram table node to update.
xmlDiffDoc
The XML Diffgram document to update.
xmlDiff
The XmlDiff Object used from the GotDotNet XmlDiff class. Performs the XML node compare.
