Infor.SST.CommandLine.CompareSchema Function
Compares and outputs a SQL diffgram schema, both as XML and SQL script files.

Private void CompareSchema (string, string, string, string, bool, bool)
  Type Name Description  
    string sourceFile The source XML file name to compare with the destination XML file.  
    string destinationFile The destination/target XML file name to compare with the source XML file.  
    string diffgramFile The XML Diffgram file name to create.  
    string SQLfile The SQL script file that will be translated from the diffgram XML schema file.  
    bool CompareSprocText A flag to cause stored procedures to be compared character by character, excluding whitespace, tabs, linefeeds and carriage returns.  
    bool Translate A flag to cause the translation method to be called, thus creating the SQL script file specified with the SQLfile param.  
  Return Description  
    void
