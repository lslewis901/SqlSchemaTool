Infor.SST.AsyncMethods.AsyncOperation.AsyncOperation Function
Initialises an AsyncOperation with an association to the supplied ISynchronizeInvoke. All events raised from this object will be delivered via this target. (This might be a Control object, so events would be delivered to that Control's UI thread.)

  Type Name Description  
    System.ComponentModel.ISynchronizeInvoke target An object implementing the ISynchronizeInvoke interface. All events will be delivered through this target, ensuring that they are delivered to the correct thread.
