Infor.SST.CommandLine.CompareColumns Function
Compares table columns from two distinct XML schema files. These files are generated by the SerializeDB method. This method is called from the CompareTables method.

Private void CompareColumns (string, System.Xml.XmlNodeList, System.Xml.XmlNodeList, System.Xml.XmlDataDocument, Microsoft.XmlDiffPatch.XmlDiff)
  Type Name Description  
    string tableName The string value representing the current SQL table object.  
    System.Xml.XmlNodeList xnlSource The XmlNodeList, a collection of source XML nodes to compare with the destination XML nodes.  
    System.Xml.XmlNodeList xnlDestination The XmlNodeList, a collection of destination/target XML nodes to compare with the source XML nodes.  
    System.Xml.XmlDataDocument xmlDiffDoc The XML Diffgram object to update.  
    Microsoft.XmlDiffPatch.XmlDiff xmlDiff The XmlDiff Object used from the GotDotNet XmlDiff class. Performs the XML node compare.  
  Return Description  
    void
