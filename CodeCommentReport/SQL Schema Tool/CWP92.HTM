Infor.SST.CommandLine.CompareTableObjs Function
Compares table indexes, table constraints, and table FK/PK references from two distinct XML schema files. These files are generated by the SerializeDB method. This method is called by the CompareTables method.

Private void CompareTableObjs (string, System.Xml.XmlNode, System.Xml.XmlNode, System.Xml.XmlNode, System.Xml.XmlDataDocument, Microsoft.XmlDiffPatch.XmlDiff)
  Type Name Description  
    string objType The type of database table related object collection to compare, i.e. indexes, constraints (check and default), and references.  
    System.Xml.XmlNode SourceTable The SourceTable XML Node containing all table object related child nodes to compare with the destination child nodes.  
    System.Xml.XmlNode DestTable The DestTable XML Node containing all table object related child nodes to compare with the source child nodes.  
    System.Xml.XmlNode DiffTable The XML Diffgram table node to update.  
    System.Xml.XmlDataDocument xmlDiffDoc The XML Diffgram document to update.  
    Microsoft.XmlDiffPatch.XmlDiff xmlDiff The XmlDiff Object used from the GotDotNet XmlDiff class. Performs the XML node compare.  
  Return Description  
    void
