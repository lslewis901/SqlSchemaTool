Infor.SST.CommandLine.SerializeDB Function
Converts sql db objects of a connected SQL server database into an XML schema file.

Private string SerializeDB (string, string, string, string, string, bool)
  Type Name Description  
    string SQLServer The string for the SQL server name to connect to.  
    string DBName The string for the SQL server database name to connect to.  
    string UID The string for the SQL server user login to use for the SqlConnections object.  
    string PWD The string for the SQL server user password to use for the SqlConnections object.  
    string SQLfile The SQL script file that will be translated from the create XML schema file.  
    bool Translate A flag to cause the translation method to be called, thus creating the SQL script file specified with the SQLfile param.  
  Return Description  
    string
