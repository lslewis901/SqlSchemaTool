Infor.SST.CommandLine.CompareObjects Function
Compares database objects (except tables and columns) from two distinct XML schema files. These files are generated by the SerializeDB method. This method is called from the CompareSchema method.

    System.Xml.XmlDataDocument xmlDiffDoc The XML Diffgram document to update.
